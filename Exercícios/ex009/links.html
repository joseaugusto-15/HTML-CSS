Links externos

Para criar links, devemos criar âncoras através da tag <a>. O pricipal atributo dessa tag é o href (hypertext reference), que cria uma referência para onde o link deve mandar

Você pode acessar o Youtube através desse link

Outro atributo útil é o hreflang, que indica qual é o idioma principal do site para onde o link está mandando

Definindo alvos

Por padrão, ao clicar em um hyperlink, o site de destino abre na mesma janela do site que contém o link, o que pode levar ao visitante do site nunca voltar. Para controlar onde o site de destino vai abrir, utiliza-se o atributo target, que suporta os seguintes valores:

  • _blank vai abrir o link em uma nova janela em branco
  • _self vai abrir o link na janela atual
  • _top Vai desfazer todos os frames e abrir o destino no navegador completo
  • _parent similar ao uso do _top em uma refência à janela mãe

O uso de frames é uma técnica quase em desuso

Indicando a natureza do destino

Para indicar a natureza do destino, utiliza-se o atributo rel, que suporta os seguintes valores:

  • next indica que o link é para a próxima parte do documento atual
  • prev indica que o link é para a parte anterior do documento atual
  • author indica que é um link para o site do autor do artigo atual
  • external indica que é um link para outro site que não faz parte do site atual
  • nofollow indica que é um link para um site não endossado, como um link pago

Links para download

Para colocar um link para download, basta fazer o link diretamente para o arquivo que se deseja efetuar o download e adicionar o atributo download com o valor configurado para o nome do arquivo a ser baixado e o atributo type para indicar ao navegador que tipo de arquivo está sendo baixado.

*Se não colocar o atributo download, o link vai apenas abrir o arquivo
